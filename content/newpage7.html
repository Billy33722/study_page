HTML
HTML (HyperText Markup Language) - это стандартный язык разметки документов для создания веб-страниц.
CSS
CSS (Cascading Style Sheets) - это язык таблиц стилей, который используется для оформления веб-страниц.
JavaScript
JavaScript - это язык программирования, который используется для создания интерактивных элементов на веб-страницах.
